Setup

  • Configure command text for triggering actions remotely. This is common for all kinds of actions.
  • Configure unique remote command number primary and secondary actions. 
  • Choose easy to remember command text & number. 

  • Separate text and number with "::" in SMS. For example, Eagle::1001
  • When GetBack receives SMS having this format, it triggers anti-theft actions as per command number.
  • If adversary has changed your SIM, GetBack would notify you on your alternate contact number. From that SMS you get sender's SIM number. Now, Send all your action commands to this number.
  • If adversary has not changed your SIM, you can send action commands to your number.
