Features

  • Assistent in phone recovery when it is stolen. It's aim is to give you at least a chance to recover and trace your phone, so that you can get it back. When this is not feasible you can still erase confidential data remotely.
  • Detection of change in SIM card to trigger Theft
  • Remote trigger via SMS by using pre-configured command text and number
  • Remotely wipe contacts & sms as part of anti theft actions
  • Capture thief's photo and send you via email. This feature is only available on device having front facing camera.
  • Detect current location and send it via SMS & Email.
  • Actions supported - Wipe contacts & SMS, Backup contacts and send via Email, Format SD card, Remove Email accounts
  • Ability to run in hidden mode
